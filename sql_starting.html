Setup

Manage your database with phpMyAdmin running on Localhost

Usage

CodeIgniter Tutorial - News section

Make a New Database

From the CodeIgniter tutorial number 2, News section, from User guide

In phpMyAdmin click new, then in the Create database, Database name field type dougtest1. Push Create button.

This made a new database named dougtest1.

Make a New Table

From CodeIgniter Tutorial - News section -- it asks you to make this database:

	CREATE TABLE news (
        id int(11) NOT NULL AUTO_INCREMENT,
        title varchar(128) NOT NULL,
        slug varchar(128) NOT NULL,
        text text NOT NULL,
        PRIMARY KEY (id),
        KEY slug (slug)
	);

Now you will use the above SQL code to create a table called news. In phpMyAdmin, select dougtest1, the click the SQL tab. Copy this text and paste it into the SQL query text window. Press the Go button.

Insert Data Into the Table

Insert items into news table. Under database dougtest1, Structure tab, click on the Table news. You should see the column names. Select the Insert tab. Click Insert tab. Under the Value column, Enter id of 1, title of "Hello World One" (don't include quotes), slug hello_1, and test of "Hi World!" (again no quotes.) Press the Go button.

You can go back to the Browse tab and see the data in your table. Go back to the Insert tab and add more rows.

Now go to the News section tutorial page and view your articles
