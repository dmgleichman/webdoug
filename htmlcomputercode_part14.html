Computer Code Elements

Some programming code using the HTML code element

The HTML code element defines a fragment of computer code

x = 5;
y = 6;
z = x + y;

Notice that the code element does not preserve extra whitespace and line-breaks. To fix this, you can put the code element inside a pre element:

	
	x = 5;	
	if (x == 77)
	{
		hello_world(x);
	}
	y = 6;
	z = x + y;

Non-breaking Space

A common character entity used in HTML is the non-breaking space: &nbsp;
A non-breaking space is a space that will not break into a new line. Two words separated by a non-breaking space will stick together (not break into a new line). This is handy when breaking the words might be disruptive.
