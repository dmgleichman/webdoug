Command Line

PHP can be run from the command line.

First add 

	c:\xampp\php
to the path

Then, to run the little example, start a Command Prompt window and enter: 

  
	c:
	cd \xampp\htdocs\webdoug\testphp
	php little.php

Sockets

Enable sockets in PHP by changing the c:\xampp\php\php.ini file.

Remove the comment ; before the line 

	extension=php_sockets.dll

UDP

UPD sockets example from BinaryTides UDP socket programming.

In one command prompt window, run: 

	php udp_server.php

In another command prompt window, run: 

	php udp_client.php

Might work better if run from two different computers?

Send and receive example

Run upd_server_two.php and udp_client_two.php for working send and receive. At least on 127.0.0.1 address.
